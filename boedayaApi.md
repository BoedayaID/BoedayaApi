# boedaya APi Application


## List Pulau api
mendapatkan list pulau di indonesia
##### URL
http://boedaya.herokuapp.com/api/pulau

##### Method
GET


## Get Detail Pulau api
mendapatkan detail pulau di indonesia


##### URL
http://boedaya.herokuapp.com/api/pulau/{id}

##### Method
GET
